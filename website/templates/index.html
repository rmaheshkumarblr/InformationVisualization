Airplane Crashes - Information Visualization Project

Technologies used

  • Python based Flask Framework
  • D3 and D3V4 for Bar Chart and other visualizations
  • HighCharts for Box Plot and Heat Map
  • d3-geomap for Choropleth
  • datamaps for respresentation of spatial data on the map
  • Google GeoCoding API to convert location to latitude and longitude
  • Geonames API to convert latitude and longitude to ISO ALPHA-2 country code
  • Pycountry library to convert ISO ALPHA-2 to ISO ALPHA-3 country code
  • Bootstrap for frontend Visualization
  • jquery and jqueryUI for Interactivity
